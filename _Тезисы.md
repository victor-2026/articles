## Post 7: ROLE OF QA (ГЛАВНЫЙ)

**Ключевые тезисы:**

### Тезис 1: Quality Gap
```
54.5% companies deployed AI in QA
44.1% disabled due to quality issues
```
- AI может генерировать тесты, но не может понять контекст
- Качество != генерация

### Тезис 2: Риск знаниеTransfer
```
"Если AI делает всё для juniors,
через 3-5 лет не будет mids/seniors —
кто будет тренировать следующее поколение?"
```
- juniors не развивают интуицию
- Не учатся искать edge cases
- Не понимают WHY за тестами

### Тезис 3: New Skill Stack

|-----|-----|
| Execute cases | Design strategy |
| Find bugs | Prevent bugs |
| Manual | AI collaboration |
| Documentation | Architecture |

### Тезис 4: Философия
```
"AI won't replace QA engineers.
But QA with AI will replace QA without it."
```
